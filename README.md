## Run Locally

Clone the project

```bash
  git clone https://github.com/Anupreet1213/Invoicer-TS.git
```

**Start the frontend with**

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the react-app 

```bash
  npm start
```

**Start the backend server with**

```bash
  cd backend
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the backend-server 

```bash
  python app.py
```
